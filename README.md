# Lane Detection Production Pipeline

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Ensure CULane dataset is at: `/Users/apple/Downloads/CULane`

## Quick Start
1. Test the pipeline: `python scripts/quick_test.py`
2. Train the model: `python scripts/train_cpu.py`
3. Run on video: `python scripts/run_pipeline.py`

## Project Structure
- `src/` - Source code
- `scripts/` - Executable scripts
- `data/` - Data directory
- `models/` - Saved models
